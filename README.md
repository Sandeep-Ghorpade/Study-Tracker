A simple console-based Java application to record daily study logs.  
You can add study entries, view all logs, generate summaries, and export data to a CSV file.

---

Features
- Add new study log (subject, hours, description)
- View all saved logs
- Summary of total study hours by **date**
- Summary of total study hours by **subject**
- Export logs to **CSV file**
- Simple menu-driven interface
